Trabalho Prático - Sistemas Distribuídos

Jogo Batalha Naval em Rede

Informações sobre o projeto:

Esse projeto foi desenvolvido como uma proposta de trabalho pratico para a disciplina de Sistemas Distribuidos do curso de Sistemas para Internet do IF Sudeste MG, tendo como objetivo a criação de um jogo de Batalha Naval com interface gráfica, onde duas pessoas poderão jogar via rede, seguindo as regras clássicas da Batalha Naval, com grids de 10x10 para cada jogador, e formato tradicional dos tipos de navios.
Também foi desenvolvido no projeto para forma de comunicação entre os participantes um chat de texto.

Desenvolvido pelos alunos do Instituto Federal do Sudeste MG:

Lucas Matos, Mateus Galdino, Matheus William, Otávio Leite e Samuel Ribeiro.

Tecnologias utilizadas:

ReactJs
Python
Flask
Mongodb
